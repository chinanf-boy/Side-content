#简单,以标题链接侧边

#how-to-use

- 设置一个 ``sider ID`` 用来显示标题

```
<ul id='sider'>
		
	</ul>
```


- 设置一个主内容 ID

>例如``side-content`` 可是自己设置

```
<div id='side-content'>
<div id="side-1"></div>
<div id="side-2"></div>
...
</div>
```

```
setSider('side-content'); //setSider('主内容ID');

or

setSider(); //默认 side-content

```

- and done ,完成
